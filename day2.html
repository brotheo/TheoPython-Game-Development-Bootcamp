‚è∞ Hour 2: Introduction to Pygame

üéØ Objective: Learn to install and run a basic pygame window with a background color.

Steps:

  • Install pygame: pip install pygame
  • Create a basic window using pygame.init()

ü™ü Sample Code ‚Äì Create Pygame Window

import pygame
import sys

pygame.init()
screen = pygame.display.set_mode((600, 400))
pygame.display.set_caption("My First Game")
background_color = (135, 206, 250)

running = True
while running:
    screen.fill(background_color)
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    pygame.display.flip()

pygame.quit()
sys.exit()

‚è∞ Hour 3: Drawing Shapes & Movement

üéØ Objective: Draw basic shapes and make them move with keyboard input.

Concepts:

  • Draw shapes: pygame.draw.rect(), pygame.draw.circle()
  • Handle keyboard input: pygame.KEYDOWN, pygame.key.get_pressed()

‚öΩ Sample Code ‚Äì Move a Circle

import pygame
import sys

pygame.init()
screen = pygame.display.set_mode((600, 400))
pygame.display.set_caption("Move the Circle")
clock = pygame.time.Clock()

x, y = 300, 200
radius = 30
speed = 5
color = (255, 0, 0)

running = True
while running:
    screen.fill((0, 0, 0))
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    keys = pygame.key.get_pressed()
    if keys[pygame.K_LEFT]:
        x -= speed
    if keys[pygame.K_RIGHT]:
        x += speed
    if keys[pygame.K_UP]:
        y -= speed
    if keys[pygame.K_DOWN]:
        y += speed

    pygame.draw.circle(screen, color, (x, y), radius)
    pygame.display.flip()
    clock.tick(60)

pygame.quit()
sys.exit()
