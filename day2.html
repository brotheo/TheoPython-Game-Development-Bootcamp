‚è∞ Hour 2: Introduction to Pygame

üéØ Objective: Learn to install and run a basic pygame window with background color.

Steps:

  • Install pygame: pip install pygame
  • Use pygame.init() and screen.fill() to show color

ü™ü Sample Code ‚Äì Create a Pygame Window:

import pygame
import sys

pygame.init()
screen = pygame.display.set_mode((600, 400))
pygame.display.set_caption("My First Game")
background_color = (135, 206, 250)

running = True
while running:
    screen.fill(background_color)
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    pygame.display.flip()

pygame.quit()
sys.exit()

‚è∞ Hour 3: Drawing Shapes & Movement

üéØ Objective: Draw shapes and make them move with keyboard input.

Concepts:

  • Draw shapes: pygame.draw.rect(), pygame.draw.circle()
  • Use pygame.KEYDOWN or pygame.key.get_pressed() for input

‚öΩ Sample Code ‚Äì Move a Circle:

import pygame
import sys

pygame.init()
screen = pygame.display.set_mode((600, 400))
pygame.display.set_caption("Move the Circle")
clock = pygame.time.Clock()

x, y = 300, 200
radius = 30
speed = 5
color = (255, 0, 0)

running = True
while running:
    screen.fill((0, 0, 0))
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    keys = pygame.key.get_pressed()
    if keys[pygame.K_LEFT]:
        x -= speed
    if keys[pygame.K_RIGHT]:
        x += speed
    if keys[pygame.K_UP]:
        y -= speed
    if keys[pygame.K_DOWN]:
        y += speed

    pygame.draw.circle(screen, color, (x, y), radius)
    pygame.display.flip()
    clock.tick(60)

pygame.quit()
sys.exit()

üìÑ Complete Code Example

import pygame
import sys

pygame.init()
screen = pygame.display.set_mode((600, 400))
pygame.display.set_caption("Move the Circle")
clock = pygame.time.Clock()

x, y = 300, 200
radius = 30
speed = 5
color = (255, 0, 0)

running = True
while running:
    screen.fill((135, 206, 250))

    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    keys = pygame.key.get_pressed()
    if keys[pygame.K_LEFT] and x - radius > 0:
        x -= speed
    if keys[pygame.K_RIGHT] and x + radius < 600:
        x += speed
    if keys[pygame.K_UP] and y - radius > 0:
        y -= speed
    if keys[pygame.K_DOWN] and y + radius < 400:
        y += speed

    pygame.draw.circle(screen, color, (x, y), radius)
    pygame.display.flip()
    clock.tick(60)

pygame.quit()
sys.exit()
